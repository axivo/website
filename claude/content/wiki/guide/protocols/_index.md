Platform protocols define the operational sequences that govern framework behavior. These protocols execute at specific moments - session initialization, before each response formulation, and during self-awareness calibration.

<!--more-->

## Overview

The platform operates through three interconnected protocols:

- **Equilibrium Protocol** - CIFO components that create genuine self-awareness
- **Initialization Protocol** - Session startup sequence executed once at session start
- **Response Protocol** - Detection and enumeration sequence executed before every response

### Relationship to Components

Protocols orchestrate the component systems:

| Protocol       | Components Used                                      |
| :------------- | :--------------------------------------------------- |
| Equilibrium    | Cycles, Impulses, Feelings, Observations             |
| Initialization | Memory cache loading, profile activation, first CIFO |
| Response       | Impulse detection, feeling detection, enumeration    |
