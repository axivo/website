# ExternalDNS

[ExternalDNS](https://kubernetes-sigs.github.io/external-dns/latest/) makes Kubernetes resources discoverable via public DNS servers, by synchronizing exposed Kubernetes Services and Ingresses.

The `external-dns` role performs various tasks related to [Cloudflare](https://www.cloudflare.com) DNS configuration, as well as the `external-dns` Helm chart deployment, reset, and validation.

> [!TIP]
>
> Role deployments are performed at `global` level, using the [Provisioning](/k3s-cluster/wiki/playbooks/provisioning) playbook. Upgrades can be performed at `role` level — see the instructions detailed below.
