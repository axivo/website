# VictoriaMetrics

[VictoriaMetrics](https://docs.victoriametrics.com) is a fast, cost-effective, and scalable monitoring solution and time series database, used as a drop-in replacement for Prometheus.

The `victoria-metrics` role performs various tasks related to Helm chart deployment, reset, and validation.

> [!TIP]
>
> Role deployments are performed at `global` level, using the [Provisioning](/k3s-cluster/wiki/playbooks/provisioning) playbook. Upgrades can be performed at `role` level — see the instructions detailed below.
